You Are The Level

A Unity project made with a team of my friends! You control the level itself by rotating it 90 degrees clockwise or anticlockwise to guide the RAT to the goal. We made this game over the course of a week, spliting into multiple teams to create the art, level design and programming for the game. It was built to be playable in Unity's web viewer, on desktop and mobile!

I was on the programming team for the game. One of my main tasks were creating the Unity TileMap for the game and adding behaviour for each tile, which allowed us to easily implement the levels that our design team made. I also made the audio system for the game and helped create the RAT character controller.

Fall Guys Inspired 3D Platformer (Individual Project)

A physics-based 3D platform made for my 4th year Advanced Game Technologies module. This game is inspired by Fall Guys, where the player must get to the end of a short obstacle course before time runs out! There are multiple levels with different objectives for the player to complete against AI controlled opponents. This coursework helped develop my C++ programming skills (particularly managing objects in memory) and debugging skills.

This project uses a Newtonian physics model and A* pathfinding for the AI characters. Some of the advanced features in this game include collision detection and resolution with spheres, AABBs, OBBs and capsules, an advanced AI with a state machine and a menu system using pushdown automata.

Dissertation - Analysis of Procedurally Generated 2D Levels

During my undergraduate degree I made a system in Unity to analyse 2D platformer levels for cohesion and difficulty. The aim was to try and procedurally create a fairly designed level that would not overwhelm the player, but still prove to be a good challenge. Research was done into procedural content generation and level design in 2D games, with a focus on Super Mario games as a basis. The project is a good base for a system that could create fair and engaging 2D levels.

Check out the repo!

BAD FOOD GAME - OpenGL Physics-Based Arcade Game

This project was made for my 3rd year Game Simulations module. It is a arcade style game utilising a Newtonian physics engine and state-based AI with flocking. Lots of varied game rules had to be implemented to make this a functioning game, such as multiple shops where the player can gain points, a life + game over system, and an invincibility balloon powerup.

Check out the repo!
